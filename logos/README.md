## ✅ シーン別おすすめ

### 名刺を作りたい
- 白い名刺 → `01_印刷・名刺_3000x1000/02_黒文字_白背景.png`
- 黒い名刺 → `01_印刷・名刺_3000x1000/03_白文字_黒背景.png`

### ホームページに貼る
- ヘッダー(明色) → `02_Webヘッダー_1200x400/01_黒文字_透過.png`
- ヘッダー(暗色) → `02_Webヘッダー_1200x400/04_白文字_透過.png`
- フッター → `03_Webコンパクト_800x300/01_黒文字_透過.png`

### SNSで使う
- プロフィール画像（明色テーマ） → `04_SNSプロフィール_1080x1080/02_黒文字_白背景.png`
- プロフィール画像（暗色テーマ） → `04_SNSプロフィール_1080x1080/03_白文字_黒背景.png`
- X(Twitter)ヘッダー → `05_SNSバナー_1500x500/03_白文字_黒背景.png`

### アプリ・サイトのアイコンに
- アプリアイコン → `07_アプリアイコン_512x512/02_黒文字_白背景.png`
- ファビコン → `08_ファビコン_256x256/02_黒文字_白背景.png`

- **フォント**: Bebas Neue Regular
- **文字間隔**: 0.18em
